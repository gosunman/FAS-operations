## 1. 예비창업패키지 2026 심사 구조

### 1.1 사업 개요 (공고문 원문 확인)
- **목적**: 혁신적인 기술과 사업모델(BM)을 보유한 예비창업자의 성공 창업 지원
- **지원금**: 평균 0.4억원 (1단계 2,000만원 + 2단계 추가, 단계별 차등)
- **선발 규모**: **총 300명** (일반분야 110명, **특화분야 190명**)
- **협약 기간**: 8개월 ('26.6 ~ '27.1 예정)
- **신청 자격**: 공고일 기준 사업자등록 및 법인 설립등기를 하지 않은 예비창업자
- **총사업비**: 정부지원사업비 100% (자기부담사업비 없음)

### 1.2 트랙 구분 (공고문 확인)
| 항목 | 일반분야 (110명) | 특화분야 (190명) |
|------|-----------------|-----------------|
| 선발 규모 | 110명 | **190명** (더 많음!) |
| 지원금 | 평균 0.4억 (차등) | 평균 0.4억 (차등) |
| 특징 | BM 혁신성 중심 | 기술 깊이 + R&D 역량 중심 |

**추천**: EIDOS SCIENCE는 **특화분야** 지원
- AI 3D 시뮬레이션 기술 + LLM 기반 자동 생성은 명확한 기술적 차별점
- 특화분야가 190명으로 일반분야(110명)보다 선발 규모 큼
- GIST 물리석사 + 네이버 개발자 = 기술 역량 입증 용이

### 1.3 평가 절차 (3단계)
1. **서류평가**: 사업계획서 서면평가 → 2배수 내외 선정
2. **인큐베이팅**: BM 구체화 교육 및 멘토링 등을 통한 역량 검증
3. **발표평가**: 발표 및 질의응답, 30분 이내

### 1.4 평가지표 (공고문 원문)

| 평가항목 | 세부 내용 (공고문) | EIDOS 어필 포인트 |
|---------|-------------------|-----------------|
| **문제인식** | 창업아이템 개발 동기(문제인식), 해결방안(필요성) | 현직 강사의 Pain Point 직접 체험, 3D 이해 한계 |
| **실현가능성** | 문제 해결을 위한 개발(개선)방법, 경쟁력 확보 방안 | MVP 이미 완성(104 테스트), 본인 학원 즉시 파일럿 |
| **성장전략** | 사업화를 위한 시장진입 전략, 자금조달 방안 | B2C→B2B→B2G, 대치동 네트워크, 11조 시장 |
| **팀(기업) 구성** | 대표자 및 고용(예정)인력의 기술역량과 노하우 | GIST 물리석사 + 네이버 개발자 + 현직 과학 강사 |

> **70점 미만 탈락**, 가점 최대 3점 (AI대학원 졸업, 창업경진대회 수상 등)

- **총 투자**: $156M (Andreessen Horowitz 리드)

- **타겟**: 대학교/고등교육
- **한계**: 고가 구독 모델, K-12 약세, 한국어 미지원, AI 자동 생성 없음

#### PhET Interactive Simulations (미국, CU Boulder)
- **시뮬레이션**: 167개
- **사용자**: 연간 1억+ 사용
- **번역**: 93개 언어, 9,313개 번역
- **연구**: 60+편 논문, Google Scholar 3,000+편 인용
- **모델**: 무료 오픈소스
- **한계**: 2D 위주, AI 미적용, 자동 교재 생성 없음, NSF 펀딩 중단($1.5M 취소)으로 지속가능성 우려

## 4. EIDOS SCIENCE 차별점 포지셔닝

### 4.1 핵심 차별화 3축

| 축 | 기존 경쟁사 | EIDOS SCIENCE |
|----|-----------|---------------|
| **시뮬레이션** | 2D 또는 사전 제작 3D | AI 기반 실시간 3D 물리/화학 시뮬레이션 |
| **콘텐츠 생성** | 수동 제작 (교사/에디터) | LLM 자동 교재 + 시험 생성 |
| **교육과정 맞춤** | 미국/유럽 커리큘럼 | 한국 교육과정 100% 대응 |

### 4.2 구체적 차별점

1. **AI 3D 물리/화학 시뮬레이션**
   - Three.js 기반 웹 3D (별도 하드웨어 불요)
   - 실시간 물리 엔진으로 실험 파라미터 조작 가능
   - 학생 수준에 따라 난이도 자동 조절

2. **LLM 기반 자동 콘텐츠 생성**
   - 시뮬레이션 연동 교재 자동 생성
   - 학생 오답 패턴 기반 맞춤 시험 문제 생성
   - 교사 업무 시간 90%+ 절감

3. **한국 교육과정 완전 대응**
   - 물리I/II, 화학I/II 전 단원 매핑
   - 대입 수능/내신 연계 문제 생성
   - 의대반 심화 실험 시뮬레이션 (현직 강사 경험 기반)

4. **즉시 검증 가능한 파일럿**
   - EIDOS SCIENCE 학원에서 즉시 실전 투입
   - 학생 성적 향상 데이터 = 사업 모델 입증

### 4.3 진입 장벽 (Moat)

- **도메인 전문성**: GIST 물리 석사 + 현직 과학 강사 → 교육 콘텐츠 품질 보증
- **기술력**: 네이버 현직 개발자 → AI + 3D 웹 기술 내재화
- **데이터**: 학원 운영으로 학생 학습 데이터 초기 확보
- **교육과정 적합성**: 한국 커리큘럼 전문화 → 글로벌 경쟁사가 쉽게 따라올 수 없는 영역

## 5. 대표자 역량 어필 포인트

### "이 문제를 풀 수 있는 유일한 사람"

| 역량 | 증빙 | 심사 연관 |
|------|------|---------|
| **과학 전문성** | GIST 물리학 석사 | 시뮬레이션 정확도 보증 |
| **개발 역량** | 네이버 현직 개발자 (6년+) | AI/3D 기술 내재화, 외주 불필요 |
| **교육 현장 이해** | 현직 과학 강사 (학원 운영) | 즉시 파일럿, 학생 데이터 확보 |
| **풀스택** | TS/Python/인프라 | 1인 MVP 개발 가능 |

**3가지 역량이 한 사람에게 있는 것이 최대 강점**
- 일반 에듀테크: 개발자 ≠ 교사 → 커뮤니케이션 비용, 도메인 오해
- EIDOS: 개발자 = 과학자 = 교사 → 빠른 이터레이션, 정확한 콘텐츠
