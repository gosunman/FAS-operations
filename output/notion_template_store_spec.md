## Executive Summary

한국의 수학/과학 학원 시장은 연간 26조 원 이상 규모이며, 약 7만 개의 학원이 운영 중이다. 그러나 이 거대한 시장을 대상으로 한 Notion 기반 운영 도구는 사실상 부재하다. Notion 공식 템플릿 갤러리에 "학교" 카테고리 13,150개 템플릿이 존재하지만, 이들은 거의 전부 **학생 자기관리용**이며, **학원 강사/원장의 운영 니즈**를 겨냥한 템플릿은 극히 드물다.

본 프로젝트는 EIDOS SCIENCE 학원 운영 경험에서 직접 검증된 워크플로우를 Notion 템플릿으로 패키징하여 판매한다. 개발 비용 제로, 도메인 전문성 활용, 향후 EIDOS SaaS 리드젠 퍼널의 입구 역할을 겸한다.

## 1. 시장 분석

### 1.1 한국 학원 시장 현황

| 지표 | 수치 | 출처 |
|------|------|------|
| 사교육비 총규모 | 26조+ 원/년 (2025) | 통계청 사교육비조사 |
| 학원 수 (전국) | ~70,000개 | 교육부 학원통계 |
| 수학/과학 학원 비중 | ~35% (약 24,500개) | 교육부 |
| 학원 강사 수 | ~250,000명 | 추정 |
| 학원당 평균 강사 | 3-5명 | 업계 평균 |
| 소규모 학원 비율 (강사 1-3명) | ~60% | 업계 추정 |

**핵심 인사이트**: 수학/과학 학원은 약 24,500개이며, 이 중 60%가 소규모(강사 1-3명)로 운영된다. 이들은 별도 관리 시스템 구매 예산이 없어 엑셀, 카카오톡, 수기 관리에 의존한다. Notion은 무료/저가로 진입 가능한 유일한 올인원 도구이다.

### 1.2 한국 학원에서의 Notion 사용 현황

**현재 상태**: 초기 도입기

- 대형 프랜차이즈 학원: 자체 LMS 사용 (아이소사이어티, 씨큐브코딩 등)
- 중형 학원 (강사 5-15명): 엑셀 + 카카오톡 업무 그룹 + 네이버 카페
- 소형 학원 (강사 1-3명): **수기 + 카톡** → Notion 도입 잠재 수요 가장 높음
- 1인 강사/과외 선생: Notion 얼리어답터 — 블로그에 "과외 Notion 활용기" 다수

**Notion 도입 장벽**:
1. 학원 원장 연령대 (40-60대)의 디지털 도구 거부감
2. "어떻게 시작해야 할지 모르겠다" → **템플릿이 정확히 이 문제를 해결**
3. 기존 엑셀 워크플로우에서 전환 비용
4. Notion 자체의 학습 곡선

**기회 요인**:
1. MZ세대 젊은 강사 급증 — Notion 친숙도 높음
2. 학부모 소통 디지털화 가속 (코로나 이후)
3. 학원 운영 효율화 니즈 증가 (인건비 상승)
4. Notion AI 기능 확장으로 학원 업무 자동화 가능성

### 1.3 기존 Notion 템플릿 마켓플레이스 분석

#### Notion 공식 갤러리
- 전체 템플릿: ~80,000개
- 학교 카테고리: 13,150개 (무료 7,459개)
- **학원 운영 전용**: 사실상 0개
- 교사용 (Teacher): 수업 계획 위주, 영어권 중심
  - Teacher Planner (A.F. Salles): 무료, 4.5/5
  - Lesson Planner (Dearsigner): 무료
  - Lesson Planner (Organized Life By Malan): $8.99
- **한국어 교육 템플릿**: 학생 플래너 위주, 학원 운영 템플릿 부재
- 가격대: 무료 ~ $8.99 (대부분 무료)

#### Gumroad
- 교육용 Notion 템플릿 존재하나 영어권 중심
- 한국 학원 특화 템플릿: 발견되지 않음
- 교사용 템플릿 가격대: $5 ~ $29
- 판매량 상위 교육 템플릿: 월 50-200건 수준

#### 크몽
- "노션 템플릿" 검색: 주로 디자인/브랜딩/개인 생산성
- 학원 관련: 거의 없음
- 가격대: ₩5,000 ~ ₩50,000
- 디지털 상품 카테고리 활성화 중

#### 아이디어스 (idus)
- 가격대: ₩1,500 ~ ₩18,000
- 주로 굿노트 스티커, 다이어리 템플릿
- 교육/학원 관련: 대학 플래너 위주, 학원 운영 없음

#### Notionery / NotionEverything 등 해외 마켓
- 가격대: $19 ~ $247
- 교육 카테고리 존재하나 학원(Academy) 특화는 없음
- 대부분 영어권 대상

### 1.4 STEM 학원 특화 템플릿의 차별점

| 구분 | 기존 교사 템플릿 | 본 프로젝트 (STEM 학원 특화) |
|------|-----------------|---------------------------|
| 대상 | 영어권 학교 교사 | 한국 수학/과학 학원 강사/원장 |
| 언어 | 영어 | 한국어 (학부모 소통 톤 반영) |
| 학제 | K-12 미국/영국 기준 | 한국 교육과정 (중등/고등, 내신/수능) |
| 운영 범위 | 수업 계획 | 수업 + 성적 + 학부모 + 운영 + 재정 |
| 학원 특수성 | 없음 | 반 편성, 정산, 학생 모집, 시험지 관리 |
| 과학 특화 | 없음 | 실험 일지, 주기율표 암기 추적, 시뮬레이션 연계 |
| 가격 | $0-$9 | ₩9,900-₩49,900 (프리미엄 포지셔닝) |

### 1.5 경쟁 분석

**직접 경쟁**: 사실상 부재
- 한국어 + 학원 운영 + Notion 템플릿의 교집합에 해당하는 제품이 현재 발견되지 않음
- 블루오션이나, 시장 크기 자체가 니치(niche)

**간접 경쟁**:
| 경쟁자 | 형태 | 가격 | 약점 |
|--------|------|------|------|
| 엑셀 성적표 | 무료 | ₩0 | 협업 불가, 모바일 불편, 데이터 사일로 |
| 학원 관리 프로그램 (아이소사이어티, 가온관리 등) | SaaS | ₩5-30만/월 | 과하게 복잡, 비용 부담, 소규모 학원엔 오버스펙 |
| 카카오톡 메모 | 무료 | ₩0 | 검색 불가, 구조화 불가, 데이터 유실 |
| 구글 시트 | 무료 | ₩0 | Notion 대비 대시보드/DB 기능 부족 |
| 네이버 밴드/카페 | 무료 | ₩0 | 데이터 관리 부적합, 학부모 소통에만 특화 |

**결론**: 본 프로젝트는 "엑셀/카톡"과 "학원관리 SaaS" 사이의 가격/기능 갭을 정확히 파고든다. ₩1-5만 원에 즉시 사용 가능한 구조화된 시스템을 제공하는 것이 핵심.

- 판매 실적 + 리뷰 확보 후 지원

### 3.2 Notion API 활용 가능성

Notion API는 템플릿 "판매" 자체에는 활용하지 않으나, 다음 용도로 활용 가능:

1. **템플릿 복제 자동화**: Notion API의 `pages.create` + `blocks.children.append`로 구매자 워크스페이스에 자동 복제
   - 현실적 한계: Notion은 "Duplicate" 링크로 충분, API 과잉 엔지니어링
   - **결론: API 미사용, Notion Duplicate 링크로 전달**

2. **데모 자동화**: API로 샘플 데이터가 채워진 데모 페이지 자동 생성
   - 구매 전 "실제 데이터로 채워진 미리보기" 제공 가능
   - **활용 가치 있음 — Phase 2에서 고려**

3. **향후 SaaS 연계**: 학생 데이터를 Notion에서 EIDOS 앱으로 마이그레이션 도구
   - **Phase 3+ EIDOS 연계 시 핵심 기능**

### 3.3 결제 시스템

#### Phase 1: Gumroad + 크몽

```
구매자 → Gumroad/크몽 결제 → 자동 이메일 → Notion Duplicate 링크 전달
```

- Gumroad: 해외 카드 / PayPal
- 크몽: 국내 카드 / 가상계좌 / 카카오페이

#### Phase 2: 자체 Toss Payments

```
구매자 → 랜딩페이지 → Toss Payments 결제
       → Webhook → 자동 이메일 (SendGrid/Resend)
       → Notion Duplicate 링크 + 사용 가이드 PDF
```

- 결제 수단: 카드, 가상계좌, 카카오페이, 네이버페이, 토스페이
- 수수료: 3.5% + ₩100/건
- 구현: Next.js API Route + Toss Payments SDK

### 3.4 디지털 상품 전달 자동화

#### 전달 방식

1. **Notion Duplicate 링크**: 구매자가 클릭 → 자신의 워크스페이스에 복제
   - 가장 간단하고 마찰 없는 방법
   - Notion 계정 필요 (무료 플랜 가능)

2. **이메일 자동 전달**:
   - 구매 완료 → 자동 이메일 발송
   - 내용: Duplicate 링크 + 사용 가이드 PDF + 동영상 튜토리얼 링크
   - Gumroad: 내장 기능
   - 자체: Resend API (월 3,000건 무료)

3. **구매 후 페이지**:
   - 결제 완료 후 리다이렉트 → 모든 링크 + 가이드 모아둔 페이지
   - 추가 구매 유도 배너 포함

#### 자동화 파이프라인 (n8n)

```
Gumroad Webhook / Toss Webhook
  → n8n
    → 구매 데이터 SQLite 저장
    → Resend 이메일 발송 (Duplicate 링크 + 가이드)
    → Notion DB에 구매자 기록
    → Slack #sales 채널 알림
    → 7일 후 자동 팔로업 이메일 (사용법 팁 + 리뷰 요청)
    → 30일 후 업셀 이메일 (번들/추가 템플릿 추천)
```

```
[인지] 블로그/인스타/카페 → 학원 운영 팁 콘텐츠 발견
  ↓
[관심] "이 강사 실제로 학원 운영하면서 이걸 쓰는구나"
  ↓
[리드] 무료 미니 템플릿 다운로드 (이메일 수집)
  ↓
[육성] 이메일 시퀀스 (5일간 3통)
  ├── Day 1: "무료 템플릿 잘 설치하셨나요?" + 사용 가이드
  ├── Day 3: "학원 운영 효율을 2배로 높이는 5가지 팁" (가치 제공)
  └── Day 5: "스타터 팩 30% 할인 — 이번 주 한정" (판매)
  ↓
[전환] 스타터 팩 또는 개별 템플릿 구매
  ↓
[업셀] 30일 후 "올인원 팩 업그레이드 — 기존 구매금액 차감"
  ↓
[리텐션] 분기별 템플릿 업데이트 알림 + 새 템플릿 출시 안내
  ↓
[추천] "동료 강사에게 추천하면 ₩5,000 할인 코드 증정"
```

## 6. EIDOS Platform 연계 전략

### 6.1 템플릿 → EIDOS SCIENCE 앱 리드젠

Notion 템플릿 스토어는 독립적 수익원이면서 동시에 EIDOS SCIENCE SaaS 플랫폼의 마케팅 퍼널 입구 역할을 한다.

```
[Notion 템플릿 구매자]
  ↓
  "성적 관리를 더 자동화하고 싶지 않으세요?"
  ↓
[EIDOS SCIENCE 앱 소개]
  - AI 시험 자동 출제
  - 3D 과학 시뮬레이션
  - 학생 학습 분석 AI
  ↓
[EIDOS 앱 무료 체험 → 유료 전환]
```

#### 구체적 연계 포인트

| 템플릿 | EIDOS 연계 기능 | 업셀 메시지 |

| T4 (시험 뱅크) | EIDOS AI 시험 출제기 | "문제를 직접 만들지 마세요, AI가 만들어 드립니다" |

| T3 (학부모 소통) | EIDOS AI 학부모 문자 | "키워드만 입력하면 AI가 문자를 작성합니다" |

### 6.2 학원 운영 템플릿 → 학생관리 SaaS 전환 경로

Notion 템플릿은 "검증된 워크플로우의 프로토타입"이다. 사용자가 Notion의 한계를 체감할 때, EIDOS SaaS가 자연스러운 업그레이드 경로가 된다.

#### Notion 한계점 (사용자가 체감하게 될 것)

| Notion 한계 | EIDOS SaaS 해결책 |
|-------------|-----------------|
| 성적 그래프 제한적 | 인터랙티브 대시보드 (Chart.js/Recharts) |
| AI 분석 불가 | 취약 단원 자동 분석, 성적 예측 |
| 시험 자동 출제 불가 | LLM 기반 문제 생성 + 난이도 조절 |
| 학부모 문자 자동화 불가 | AI 문자 초안 + 원클릭 발송 |
| OCR 채점 불가 | 답안지 사진 → 자동 채점 + 점수 기록 |
| 데이터 분석 제한적 | 학생별/반별/학기별 종합 분석 |
| 모바일 앱 UX 부족 | 전용 모바일 앱 (React Native) |
| 다중 사용자 권한 제한 | 원장/강사/학부모 역할별 접근 |

#### 전환 경로 설계

```
Phase 1 (현재): Notion 템플릿 판매
  - 사용자 확보 + 니즈 검증 + 이메일 리스트 구축

Phase 2 (3개월 후): 템플릿 사용자 대상 설문
  - "가장 불편한 점은?" "추가로 원하는 기능은?"
  - → 제품 로드맵 우선순위 결정

Phase 3 (6개월 후): EIDOS SaaS 베타
  - 템플릿 구매자 → 무료 베타 초대
  - Notion 데이터 → EIDOS 마이그레이션 도구 제공
  - 가격: ₩29,000/월 (학원당)

Phase 4 (12개월 후): EIDOS SaaS 정식 출시
  - 템플릿은 무료로 전환 (SaaS 리드젠 전용)
  - 또는 템플릿 구매자 → SaaS 3개월 무료 전환
```

#### 데이터 마이그레이션 경로

```typescript

// Notion API로 기존 데이터 읽기 → EIDOS DB에 입력

// 사용자 워크플로우:
// 1. EIDOS에서 "Notion에서 가져오기" 버튼 클릭
// 2. Notion OAuth 인증
// 3. 학생 DB, 성적 DB, 수업 DB 자동 감지
// 4. 매핑 확인 → 가져오기 실행
```
